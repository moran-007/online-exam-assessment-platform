# 题目批量录入模板

更新时间：2026-07-04

## 使用方式

在“题目导入”页面中切换到“批量导入”，分别填写“内容”和“答案”。多道题之间使用单独一行 `---` 分隔。

页面说明：

- “单题导入”和“批量导入”放在同一个页面，可通过页签切换。
- 页面顶部选择课程、知识点、批次标签和发布设置；标签可搜索已有标签，也可直接输入新标签。
- 左侧填写题目内容、答案配置；右侧实时预览解析结果。
- 右侧会实时展示 Markdown 题干预览、选项、正确答案、解析和解析表。
- 批次标签和批次知识点会追加到本次导入的所有题目上，题目块内的 `标签`、`知识点` 字段仍然保留。
- 单题导入可直接在题干、选项、解析中粘贴图片，系统会自动上传并插入 Markdown 图片引用。
- 批量导入可通过页面“附件”工具上传图片、PDF 或其他文件，再把生成的 Markdown 链接插入到当前批量模板中。
- 点击“解析预览”会检查格式；只有没有错误的题目才能导入。
- 点击“重复检测”会检查题库中已有题目；批量导入时重复题和冲突题会默认跳过，其余题目继续导入。
- 解析预览表支持“移除”单道题；移除只影响本次导入队列，不会改写原始文件或模板文本。

字段说明：

- `标题`：必填。
- `题型`：支持 `单选题`、`多选题`、`判断题`、`填空题`、`简答题`、`编程题`、`材料题`、`材料/组合题`、`大题/组合题`、`文件上传题`、`Scratch 项目题`、`Arduino 项目题`，也支持接口值如 `single_choice`。
- `难度`：1 到 5。
- `分值`：数字。
- `标签`：可选，多个标签用 `,`、`，`、`、`、`|` 或 `/` 分隔；导入时不存在的标签会自动创建。
- `知识点`：可选，多个知识点用 `,`、`，`、`、`、`|` 或 `/` 分隔；需与当前课程下已有知识点名称一致。也可以在页面顶部批量选择同一批次知识点。
- `题干`、`选项`、`解析`：支持 Markdown 和代码块。
- 图片：使用 Markdown 语法，例如 `![题目图片](/uploads/question-assets/example.png)`；该路径是逻辑资源标识，页面会自动通过鉴权或短时签名接口读取，请优先使用上传/粘贴功能生成。

Excel 模板新增材料/组合题列：

- `题目标识`：同一 Excel 内的临时标识，例如 `M-001`、`Q-001`，用于父子题关联，导入后不会作为题目 ID。
- `父题标识`：子题填写所属材料/组合题的 `题目标识`；普通题和父题留空。
- `子题分值`：子题在材料/组合题中的独立分值；不填或非正数时沿用子题自己的 `分值`。
- `子题顺序`：子题在材料/组合题下的展示顺序；不填时按 Excel 行顺序。

单题导入中，`大题/组合题（材料/多问）` 支持直接在父题表单内联添加多个简答、填空、选择等小题；填空小题与普通填空题一样逐空填写答案。Excel 批量材料/组合题采用两遍导入：先导入非材料子题，再按 `父题标识` 组装父题。当前只支持单层组合，材料/组合题不能继续作为另一个材料/组合题的子题。

答案配置说明：

- 每行一个答案，推荐按题号配置：`1. B`、`2. A,B`。
- 也可以按标题配置：`Python 输出结果：B`。
- 单选/判断填一个选项，如 `B`；多选用英文逗号分隔，如 `A,B`；填空题可用逗号分隔多个可接受答案。
- 为兼容旧模板，题目内容中仍可写 `答案：B`，但推荐统一放到答案区。
- 点击“解析预览”会先校验格式。若有问题，预览表会标出“第几题”以及错误原因，修正后再导入。

批次标签示例：

```txt
GESP, 2026春季, 阶段测验
```

若题目块内写了 `标签：Python,代码阅读`，最终标签会合并为 `GESP`、`2026春季`、`阶段测验`、`Python`、`代码阅读`。

## 单选题示例

内容：

````md
标题：Python 输出结果
题型：单选题
难度：1
分值：2
标签：Python,代码阅读
知识点：变量与表达式
题干：
阅读代码，输出结果是什么？

```python
print(1 + 2)
```
选项：
A. `1`
B. `3`
C. `12`
D. `None`
解析：
表达式 `1 + 2` 的结果是 `3`。
````

答案：

```txt
1. B
```

## 图片题示例

单题导入时可以直接粘贴图片，系统会自动上传并插入图片引用。批量导入时先用页面“图片”按钮上传，再把生成的 Markdown 引用插入题干或选项。

内容：

````md
标题：图片选择题示例

难度：1
````

## 常见错误示例

下面这类内容可以被“解析预览”拦截，并提示具体题号：

```md
标题：格式错误示例
题型：单选题
难度：1
分值：2
标签：格式校验
题干：
这道题没有配置答案，应该被指出问题。
选项：
A. 选项一
B. 选项二
```

对应错误：单选题没有且只有一个正确选项，需要在答案区填写 `1. A` 或 `1. B`。

## Hydro/Markdown 选择题转换建议

`P_list\611` 这类题目可转换为本模板：

- 每个编号题转成一个题目块。
- `{{ select(n) }}` 删除，答案统一放到答案区。
- Markdown 代码块保留在题干或选项中。
- 原始标签如 `GESP` 可写入 `标签：GESP,P_list_611,选择题`。
