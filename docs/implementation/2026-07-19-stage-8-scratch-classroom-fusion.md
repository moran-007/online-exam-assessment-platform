# Stage 8 Scratch 课堂融合实施记录

日期：2026-07-19  
状态：完成，Stage 8 退出门禁通过

## 交付范围

- 主平台新增 Scratch 模板、课次任务、学生作品、不可变版本、教师批阅、判定任务和幂等回调模型。
- 教师可在教学记录抽屉内上传经结构校验的 `.sb3`、绑定并发布任务、查看作品、下载历史版本、追加批阅和重试失败判定。
- 学生可在学习门户下载模板、创建作品、保存新版本和提交；家长仅能查看关联学生已有版本、判定与点评。
- 作品、缩略图和迁移文件进入统一对象存储。下载前同时校验大小和 SHA-256；本地存储的 `stat` 已返回真实哈希。
- Scratch Adapter 使用异步提交、超时、有限重试、租约恢复、幂等键和 HMAC 回调；外部运行时未配置或不可用时只更新判定任务状态，不删除或覆盖本地作品。
- 学生总结只接入已发布任务中的作品状态、版本数量、判定结果和教师批阅，不读取项目内容推断能力，也不使用外部运行时身份。

## 数据与权限

- 新增 11 项细粒度权限，教师、学生、家长分别获得维护/发布/批阅、本人保存/提交、关联学生只读权限。
- 模板下载按已发布任务和班级成员校验；作品下载按本人、关联家长或任教范围校验。
- 数据库触发器禁止修改或删除作品版本、批阅和回调记录。
- `.sb3` 校验真实 ZIP、`project.json`、舞台结构、路径穿越、嵌套压缩包、素材数量/大小和异常压缩率。
