# 阶段 6：教学记录、资源与学习门户实施记录

## 完成结论

阶段 6 已完成。主平台已形成“教师编辑—提交—发布—学生/家长查看”的教学记录闭环，并完成 worker_01 历史教学记录和附件的可追溯迁移。

核心结论：

- 每个课次至多一份当前教学记录，所有保存、状态切换和附件变化都追加不可变版本。
- 已发布内容一旦编辑或变更附件，立即退回草稿并清除发布信息。
- 教师内部字段不进入学生或家长响应；服务端数据范围校验不是前端隐藏按钮。
- 家长只能选择已建立有效关系的学生，未关联家长不能构造学生 ID 越权。
- 附件复用 `FileAsset` 和 `ObjectStorage`，下载前核对对象存在性和文件大小。
- 发布会通知当前班级学生和已关联家长，并写入审计日志。
- 学习门户统一展示已发布课次、考试结果和已发布 AI 总结；AI 不可用不影响课次和考试查询。

## 领域模型与状态机

本阶段新增：

- `LessonRecord`：课次当前记录，包含公开字段、教师内部字段、状态和乐观并发版本号。
- `LessonRecordVersion`：保存草稿、提交、发布、附件增删和历史导入的不可变快照。
- `LessonAsset`：教学记录与统一 `FileAsset` 的关联，区分 `INTERNAL` 和 `LEARNER`。

状态流：

1. 首次保存创建 `DRAFT`。
2. `DRAFT` 可提交为 `SUBMITTED`。
3. `SUBMITTED` 可发布为 `PUBLISHED`。
4. 任意状态下继续编辑，或增删附件，都会生成新版本并回到 `DRAFT`。
5. 提交和发布使用 `id + version + status` 条件更新；并发冲突不会静默覆盖。

数据库触发器禁止更新或删除 `lesson_record_versions`。当前记录可以继续演进，但历史证据不能被原地改写。

## 页面与接口

教师在“教学运营”课次行打开“教学记录”抽屉，可编辑公开内容和内部内容、保存、提交、发布、上传附件并查看版本历史。

学生和家长新增“学习门户”，包括：

- 已发布课次内容和公开附件；
- 考试记录，继续服从原有成绩发布范围；
- 已发布 AI 总结；
- 家长关联学生选择器。

主要接口：

- `GET /lesson-records`、`GET /lesson-records/:sessionId`
- `PUT /lesson-records/:sessionId/draft`
- `POST /lesson-records/:sessionId/submit`
- `POST /lesson-records/:sessionId/publish`
- `GET /lesson-records/:sessionId/versions`
- `POST /lesson-records/:sessionId/assets`
- `DELETE /lesson-records/:sessionId/assets/:assetId`
- `GET /lesson-records/:sessionId/assets/:assetId/content`
- `GET /learning-portal/students`
- `GET /learning-portal/students/:studentId`

OpenAPI 文档和前端生成客户端已同步。

## 附件安全与浏览器兼容

上传限制为单文件 50 MB，允许图片、PDF、Office、ZIP、文本和 Scratch `.sb3`。服务端同时检查扩展名、MIME 和内容签名；可执行文件、伪装类型、空文件和大小不一致文件会被拒绝。

Scratch 仅在扩展名为 `.sb3` 且内容具有 ZIP 签名时接受。对象采用随机键存储，原文件名只作为经过清理的显示名称。

真实浏览器验收发现并修复两项编码问题：

- Multer 将 UTF-8 中文文件名暴露为 Latin-1 字符时，使用可逆转换归一化；真实 Latin-1 名称不会被破坏。
- 文本响应统一声明 UTF-8；Blob 预览增加只影响预览的 UTF-8 BOM，下载仍使用原始字节，SHA-256 不变化。

字符集处理已提取为公共 HTTP 工具，同时用于教学附件、题库附件和导出响应。

## 运维与回滚

- 业务回滚只撤下入口或权限，不修改不可变历史版本。
- 对象写入失败不会创建数据库附件；数据库事务失败会清理已写入对象。
- 对象丢失或大小变化时返回明确错误，不以数据库元数据伪装成功。
- 迁移失败会将 `MigrationRun` 标记为失败并保留原因；同一指纹可安全重跑。
- 学习门户不依赖模型调用，AI 供应商不可用时课次、考试和已发布历史仍正常。

阶段 6 退出条件全部满足，可以进入阶段 7。后续如果公开字段、家长关系或附件受众规则变化，必须重新执行学生、关联家长和未关联家长三类浏览器验收。
