# 外部编程题导入与测评说明

## 外部题导入

- 编程题可以只填写远端题号，也可以直接粘贴题目地址。选择站点后，系统会自动推断平台、域、题目链接和同站点录入账号。
- 题目来源会写入题目标签，例如 `外部编程题`、`tarjanoj.com`，便于题库筛选和后续按账号提交。
- 修改题号时，如果旧链接指向其他题目，前端会自动丢弃旧链接，避免继续拉取上一题。

## 账号与提交

- 管理员、教师、助教和学生都可以在个人信息页维护自己的外部账号。
- 外部账号管理页顶部会单独列出“我的账号”，超级管理员不需要在全量账号列表里查找自己的绑定。
- 录入账号和学生提交账号都按同站点匹配，`www.tarjanoj.com` 和 `tarjanoj.com` 会视为同一站点。
- 提交时统一使用账号绑定的站点地址生成提交路径，避免跨域提交或学生无权限访问。
- 学生作答时必须选择当前题目来源站点下的账号；没有同站点账号时不允许提交。
- 教师也可以在题库作答弹窗提交代码，用于验证外部题绑定和评测链路是否正常。

## 测评语言

- 远端 Hydro/Tarjan 题会尽量从题目配置和页面语言下拉框提取真实语言 ID。
- 例如 Tarjan B2002 会返回 `cc.cc14o2` 和 `py.py3`，分别对应远端的 `C++14(O2)` 和 `Python 3`。
- 代码编辑器顶部显示的是评测语言名称；内部语法高亮仍会使用 `cpp`、`python` 等高亮器语言。

## 填空题

- 题干中的 `____` 是学生看到的填空横线，也会作为自动识别空位数量的标记。
- 单题导入和题库编辑都支持手动“增加空位”和“插入题干空位”，不需要老师手写“第1空/第2空”。
- 多个正确答案可以在同一个空位中用逗号分隔。

## 导入表单体验

- 单题导入成功后会记住本次题型；下次清空或重新进入时默认使用上一次成功导入的题型。
- 标题输入框在单题导入中单独占一整行，减少漏改示例标题的风险。
