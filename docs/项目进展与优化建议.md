# 项目进展与优化建议

更新时间：2026-06-29

## 一、本次参考说明

本次只参考了本地历史项目的工程组织和启动方式，没有复用其业务代码。

参考价值：

1. Vue 前端项目样例
   - 参考点：前端使用 Vite + Vue + Element Plus；根目录放置 `.bat/.ps1` 启动文件。
   - 应用到本项目：本项目新增 `frontend` 子目录，并提供 `启动平台.bat`、`停止平台.bat`。
2. 前后端分离项目样例
   - 参考点：前后端分目录、统一工作区、日志文件分离。
   - 应用到本项目：本项目新增 `runtime` 日志目录和前后端独立启动进程。

## 二、当前已实现

### 2.1 后端基础框架

- NestJS + Prisma + PostgreSQL。
- 统一接口前缀：`/api/v1`。
- 统一返回格式：`{ code, message, data }`。
- JWT Access Token + Refresh Token。
- Refresh Token 哈希入库，可刷新、可退出失效。
- 全局登录守卫、角色守卫、权限守卫。
- 审计日志基础服务。

### 2.2 登录与扩展预留

- 当前实现账号密码登录 provider。
- 已新增 `auth_accounts` 表，用于后续接入其他平台共同登录。
- 后续接入企业微信、钉钉、学校统一认证时，建议新增独立 provider，不改现有密码登录流程。

### 2.3 核心业务闭环

- 课程管理。
- 课程管理支持单条新增、编辑、启用/停用、关键词查询。
- 课程管理支持删除；删除采用软删除，并在课程仍有关联知识点、题目、试卷或考试时拦截，避免破坏历史数据。
- 课程支持按钮弹窗批量导入，模板简化为 `课程名称 | 第几节课 | 描述`，课程编码由系统自动生成或沿用已有课程编码。
- 知识点树管理。
- 课程与知识点已整理到同一管理页，按课程折叠展示课程下的知识点树，便于按 Python、C++ 等课程维护分支、循环等子集。
- 知识点支持新增、编辑、启用/停用、删除；删除叶子知识点时会解除题目关联，有子知识点时会提示先移动或删除子节点。
- 知识点支持按钮弹窗批量导入，先选择所属课程，再按模板 `知识点名称 | 父级知识点名称 | 排序` 导入；知识点编码由系统自动生成。
- 批量导入会先校验所属课程、父级引用、同批次重复项；已存在课程或知识点会按名称更新，新的会自动创建。
- 标签管理。
- 标签支持新增、编辑、启用/停用、删除；删除标签不会删除题目，只会解除题目与该标签的关联。
- 题库管理：创建、查看、编辑、发布、取消发布、隐藏、显示、软删除。
- 题库查询：支持按课程、标签、状态、关键词筛选。
- 题库查询支持按课程下的知识点筛选。
- 题库公开访问：访客可查看公开题库，学员登录后可进入题目作答并即时判分。
- 被已安排或进行中考试占用的题目会从公开题库隐藏，管理员题库中仍可见，但列表只显示“隐藏”标记，不直接展示占用考试名称；点击标记可跳转到考试管理定位对应考试。
- 支持题型：单选、多选、判断、填空、简答、编程、材料、文件上传、Scratch 项目、Arduino 项目。
- 题目录入支持 Markdown 与代码块，题干、选项、解析均可渲染预览。
- 单题录入支持自定义标签；不存在的标签会自动创建。
- 多选题选项不再固定为 4 个，可在录入时继续增加或删除选项。
- 题库与题目导入已拆分为两个页面：题库用于查询和维护，题目导入集中放置单题导入与批量导入。
- 题库页默认不显示题目录入区域，也不在顶部提供导入按钮；导入统一从左侧“题目导入”菜单进入。
- 题目导入页内包含“单题导入 / 批量导入”两个页签，两种方式共用课程、标签和导入后发布设置。
- 单题导入支持 Markdown 题干、代码块、动态选项、自定义标签、实时预览和导入后发布。
- 单题导入可选择所属知识点，支持在题干、选项、解析中粘贴图片自动上传并插入 Markdown 图片引用。
- 单题导入已接入重复/冲突检测；输入变化后会静默检测，导入前若发现重复、相似或冲突会二次确认，避免误建重复题。
- 题目导入支持 Markdown 模板批量录入，可一次导入多道题并自动发布。
- 批量录入支持题目内容与答案配置分离，答案可按题号或标题单独维护。
- 批量导入支持批次级标签和批次级知识点，同一批次可统一追加相同标签和知识点。
- 题目导入页顶部课程、知识点、标签、发布、图片工具与单题基础字段采用横向合并布局，便于对照题目输入。
- 题目导入标签不再展示快捷提示，统一由用户搜索已有标签或直接输入新标签。
- 填空题录入支持配置是否区分大小写、是否区分首尾空格，并写入判题配置。
- 批量导入支持多图片上传，上传后可修改图片引用名，再插入到当前单题或批量模板文本中；题目块内也可使用 Markdown 图片语法。
- 批量录入会先校验格式，并在预览表中标注具体第几题、哪个位置有问题。
- 批量导入页面会实时渲染题干 Markdown、代码块、选项和解析，解析后可逐题查看导入结果。
- 批量导入预览表支持手动移除当前批次内不需要导入的题目；重复/冲突检测后，导入时会自动跳过重复题和冲突题，其余题目继续导入，相似题默认提示但不拦截。
- 批量录入模板文档：`docs\题目批量录入模板.md`。
- 题目快照与版本记录。
- 试卷管理：创建、添加题目、规则组卷、预览、发布。
- 试卷管理主界面已整理为“试卷库”列表，创建试卷使用弹窗，点击试卷行默认打开预览抽屉，调整题目与按条件组卷收纳到调整抽屉。
- 试卷组卷后支持预览题目内容、选项、正确答案与解析。
- 试卷支持按课程/大类、知识点、标签、题型组合筛选批量加入题目，也可按条件随机抽取指定数量题目加入试卷。
- 管理员和教师可进入试卷试答页面，按整卷作答并查看红绿判分，用于检查题目和答案是否完全正确。
- 草稿试卷支持继续添加题目、删除题目、调整题目分值和分区。
- 试卷题目支持上移、下移，便于手动控制题目顺序。
- 试卷基础信息支持修改名称、课程、时长和乱序设置。
- 试卷题目随机、选项随机已在学生进入考试生成试卷实例时生效；同一学生实例内顺序固定。
- 试卷支持按课程折叠分类查看，便于后续扩展二级菜单和归档管理。
- 试卷中的题目显示快照支持单独编辑题干、选项显示内容与解析；该操作不会修改原始题库题目。
- 试卷显示快照编辑设有安全边界：关联考试运行/结束、已生成学生实例或已有提交记录时禁止编辑，避免正式答题后题面被追改。
- 已发布试卷支持取消发布后继续调整。
- 已发布/历史试卷支持复制为新草稿，便于在不影响原试卷和历史答卷的情况下继续调整。
- 考试管理支持批量开始、结束、暂停等状态变更，并返回无需用户点击的结果提示。
- 题库在编辑模式下才允许修改题目状态、复制或删除；退出编辑模式后点击题目按学生答题流程进入。
- 题库支持批量删除；公开题库中登录学员可批量加入错题本。
- 学生考试列表增加当次答题和比赛结束倒计时提示；进入正式答题页后保持专注模式，仅显示答题所需内容。
- 管理员与教师账号均可查看已生成/已发布试卷。
- 考试管理：创建、编辑、发布、取消发布、删除、成绩列表、基础统计。
- 考试管理主界面已整理为考试列表，创建/编辑使用弹窗，点击考试行默认打开考试预览抽屉，排名/成绩从独立抽屉查看。
- 考试支持修改名称、试卷、课程、起止时间、考试时长、次数限制、答案与分数显示策略。
- 考试结果开放支持分类控制：小题得分、题干内容、学生作答、对错结果、正确答案、解析可分别配置。
- 默认规则：考试未结束或学生可提交次数未用完时，只开放分数，不开放解析和正确答案；管理员可按分类提前开放。
- 考试支持公告配置，学生进入考试前需要阅读并确认。
- 考试公告已新增独立公告表与阅读记录表；新建/更新考试公告会写入公告版本，学生确认后会记录阅读时间。
- 学生端：我的考试、进入考试、保存答案、批量保存、提交、查看结果、错题本。
- 学生进入考试后使用专注答题模式，不显示左侧导航和顶部导航栏。
- 学生考试次数限制已落实；达到次数上限后不会创建新的正式答题记录。
- 学生端新增个人信息页，显示姓名、账号、身份、角色和权限数量，便于确认试卷归属。
- 客观题自动判分：单选、多选、判断、填空。
- 错题自动收录。
- 错题本已调整为个人题库形态，支持手动加入公开题、移出题目、随机抽题弹窗作答。
- 错题本答对后自动标记掌握并从当前错题列表隐藏；手动移出会标记为忽略。
- 教师/管理员可进行 trial/test 试答，用于检查题目正确性，不计入正式成绩、提交记录或排名。
- 教师/管理员可选择真实学生进行模拟作答，用于验证学生流程；该模式会按学生身份产生记录，和 trial/test 明确区分。
